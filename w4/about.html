Understanding the output from httpbin

The primary use of httpbins is to help developers inspect what data is being sent, and to test form submissions, headers, and other http request details.

If you submit the form, httpbin will respond by "echoing" back the form data that was sent in the request. The output you see will be a JSON response that shows the data that was sent in the request body.
